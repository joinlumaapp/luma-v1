# 📌 Core Product Decisions

1. 100% compatibility does not exist.
2. LUMA is NOT a random swipe app.
3. There is no strict “serious vs casual” separation.
4. Premium does not modify compatibility score.
5. Compatibility Room opens only after mutual match.
6. Relationship Mode blocks new matches.
7. Couples Club is not a flirting environment.
8. Score visibility range is fixed between 47–97.
