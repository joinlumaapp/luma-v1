# LUMA Monetization Model

## CRITICAL RULE
**No feature is fully locked. Every feature is accessible to ALL users. Packages only change QUANTITIES and LIMITS.**

The goal: Give everyone a taste → Premium users want more → Supreme users get unlimited.

## Revenue Streams Summary
1. **Subscriptions** — Premium (499₺/ay) and Supreme (1.199₺/ay)
2. **Jeton Sales** — In-app token purchases
3. **Boost Purchases** — Profile visibility boost via jeton
4. **Rewarded Ads** — Free users watch ads for jeton/temporary features (Google AdMob)

## Founder Badge
- First 777 users get special "Kurucu" (Founder) badge
- Badge is permanent and cannot be purchased
- Shows on profile as exclusive recognition
- Includes: launch discount on first subscription

## Pricing Strategy Notes
- Premium should feel like "I can use everything but want more"
- Supreme should feel like "I own this app, no limits"
- Ücretsiz should feel like "This app is amazing, I need to upgrade"
- NEVER frustrate free users — let them taste everything, then limit quantities
- Jeton costs should feel achievable through missions for patient users, or purchasable for impatient users
